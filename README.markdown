# Class-Implant

Manipulating mixin and inheritance outside of packages

## INSTALLATION

To install this module type the following:

   perl Makefile.PL

   make

   make test

   make install

## DEPENDENCIES

This module requires these other modules and libraries:

  Exporter

  Class::Inspector
